# Student Management System

A web-based Marks Management System developed using Spring Boot, Thymeleaf, and MySQL for the ICT3122 Advanced Programming Practicum module at the University of Ruhuna.

## 🚀 Features

* Student Management
* Marks Management
* Automatic Total & Grade Calculation
* Student Record Updates
* Safe Delete System
* Responsive Web Interface

## 🛠 Technologies Used

* Java
* Spring Boot
* Thymeleaf
* MySQL
* Spring Data JPA
* HTML/CSS

## 🏗 Architecture

MVC Architecture using:

* Model
* View
* Controller
* Service Layer

## ▶️ Run the Project

1. Configure MySQL database
2. Update `application.properties`
3. Run the Spring Boot application
4. Open:

```text
http://localhost:8080
```
